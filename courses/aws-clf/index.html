Faktorer som påvirker AWS-pris

Beregning
EC2 faktureres per sekund (min 1 min). Lambda per antall kall + GB-sekunder.
Lagring
S3: per GB lagret + per forespørsel. EBS: per GB provisionert per måned.
Datatrafikk
Innkommende data er gratis. Utgående til internett koster per GB (første 100 GB/mnd gratis).
Huskeregel: Forutsigbar og langsiktig → Reserved/Savings Plans. Uforutsigbar → On-Demand. Tåler avbrudd → Spot. Compliance + lisensiering → Dedicated Hosts.

AWS Shared Responsibility Model

AWS er ansvarlig for sikkerheten av skyen. Kunden er ansvarlig for sikkerheten i skyen.

☁️ AWS sitt ansvar

  • Fysisk infrastruktur (datasentre)
  • Maskinvare (servere, nettverk)
  • Hypervisor og virtualisering
  • Managed services (RDS, S3 infrastruktur)
  • Global nettverksinfrastruktur
  • Fasilitetssikkerhet

👤 Kundens ansvar

  • Operativsystem og oppdateringer (EC2)
  • Applikasjonskode og konfigurasjon
  • IAM-brukere, roller og tilganger
  • Data (kryptering, backup)
  • Nettverkskonfigurasjon (Security Groups)
  • Klientside og endepunktsikkerhet
Eksamentips: "Security OF the cloud" = AWS. "Security IN the cloud" = kunden. For managed services som RDS tar AWS mer ansvar (OS-patching), mens for EC2 tar kunden alt ansvar for OS og oppover.

Variert ansvar per tjenestetype

IaaS (EC2)
Kunden har mest ansvar: OS, middleware, runtime, data, applikasjon.
PaaS (Elastic Beanstalk)
Delt: AWS håndterer plattformen, kunden håndterer kode og data.
SaaS (Amazon Chime)
AWS håndterer nesten alt. Kunden er kun ansvarlig for egne data og tilganger.
